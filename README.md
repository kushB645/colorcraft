# 🎨 ColorCraft — Color Palette Generator  

A modern and interactive **color palette generator** built to help developers and designers quickly create, explore, and use beautiful color combinations for their projects.

---

## 🚀 Live Demo  
🔗 https://colorcraft01.vercel.app/

---

## 📌 Features  

- 🎨 Generate beautiful color palettes  
- ⚡ Fast and responsive UI  
- 📋 Copy color codes instantly  
- 🌈 Clean and minimal design  
- 📱 Fully responsive (mobile + desktop)  
- ✨ Smooth interactions and hover effects

## ⚙️ Installation & Setup  

### 1. Clone the repository  
git clone https://github.com/your-username/colorcraft.git  

### 2. Navigate to project folder  
cd colorcraft  

### 3. Install dependencies  
npm install  

### 4. Start development server  
npm start  

### 5. Build for production  
npm run build
